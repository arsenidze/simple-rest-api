# simple-rest-api

## Project overview

Skeleton of simple REST Api using `Vue.js` and `Node.js`

![main_page](/images/main_page.png)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

To run on `localhost:3000`:

```
npm run dev
```
